## How It Works

1. **Enter your username** on the welcome screen
2. **Listen to up to 3 progressive audio clues** for a randomly generated location:
   - 🔊 **Stage 1 — Ambient Sounds** (3.0× score multiplier)
   - 🎵 **Stage 2 — Regional Music** (1.8× multiplier)
   - 🗣️ **Stage 3 — Spoken Phrase** in the local language (1.0× multiplier)
3. **Place a pin** on the interactive 3D globe or Google Maps view
4. **View your score** — the closer and earlier you guess, the higher you score
5. **Replay any unlocked clue** — all previously heard audio stays accessible
6. **Compete on the leaderboard** against other players

## Scoring

Uses exponential decay so nearby guesses are rewarded much more than far-off ones. Guessing at an earlier stage gives a significantly higher score. Maximum possible score: **15,000**.

```
score = min(15000, 5000 × e^(-distance_km / 2000) × stage_multiplier)
```

| Stage | Multiplier | Max Score |
|---|---|---|
| 1 — Ambient | 3.0× | 15,000 |
| 2 — Music | 1.8× | 9,000 |
| 3 — Language | 1.0× | 5,000 |

After each round you receive a tier badge: **Perfect** (8000+) · **Amazing** (5000+) · **Great** (3000+) · **Good** (1000+) · **Keep Trying**

## Audio Generation

Each round generates three AI audio clues in parallel using **ElevenLabs**:

- **Ambient** (10s) — 3-4 loud, distinctive non-vocal sounds that identify the country (e.g., espresso machines, rickshaw bells, cicadas)
- **Music** (15s) — Country-specific musical style with regional instruments
- **Language** (one sentence, 15-25 words) — A natural conversational sentence in the local language, spoken by a dynamically generated voice matching the region's accent

Location prompts are generated by **Backboard AI** (Gemini-powered) with strict rules: no city names, landmark names, or station names in any audio prompt to prevent trivial identification.

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Setup

```bash
git clone https://github.com/AdrianLuk12/LiHacks.git
cd LiHacks
npm install
cp .env.example .env.local
```

Add your API keys to `.env.local`:

```env
BACKBOARD_API_KEY=your_backboard_api_key
BACKBOARD_API_URL=https://app.backboard.io/api
ELEVENLABS_API_KEY=your_elevenlabs_api_key
NEXT_PUBLIC_GOOGLE_MAPS_API_KEY=your_google_maps_api_key
```

### Run

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

│   ├── LoadingScreen.tsx

## API Routes

| Route | Method | Description |
|---|---|---|
| `/api/game/generate` | POST | Generate a random location with 3 audio clues |
| `/api/game/guess` | POST | Submit map coordinates, receive score and actual location |
| `/api/leaderboard` | GET | Fetch top 100 scores |
| `/api/leaderboard` | POST | Save a score to the leaderboard |
