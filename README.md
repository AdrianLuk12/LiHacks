# 🎧 SonicGuessr

An audio-based geography guessing game where players identify real-world locations by listening to AI-generated sound clues. Built for LiHacks.

## How It Works

1. **Enter your username** on the welcome screen
2. **Listen to up to 3 progressive audio clues** for a randomly generated location:
   - 🔊 **Stage 1 — Ambient Sounds** (3.0× score multiplier)
   - 🎵 **Stage 2 — Regional Music** (1.8× multiplier)
   - 🗣️ **Stage 3 — Spoken Phrase** in the local language (1.0× multiplier)
3. **Place a pin** on the interactive 3D globe to submit your guess
4. **View your score** — the closer and earlier you guess, the higher you score
5. **Replay any unlocked clue** — all previously heard audio stays accessible
6. **Compete on the leaderboard** against other players

## Scoring

**Formula:** `min(15000, 5000 × e^(-distance_km / 2000) × stage_multiplier)`

Uses exponential decay so nearby guesses are rewarded much more than far-off ones. Guessing accurately at an earlier stage gives a significantly higher score. Maximum possible score is 15,000 (perfect guess at Stage 1).

| Stage | Multiplier | Max Score |
|---|---|---|
| 1 — Ambient | 3.0× | 15,000 |
| 2 — Music | 1.8× | 9,000 |
| 3 — Language | 1.0× | 5,000 |

## Audio Generation

Each round generates three AI audio clues using **ElevenLabs**:

- **Ambient** (10s) — 3-4 loud, distinctive non-vocal sounds that identify the country (e.g., espresso machines, rickshaw bells, cicadas). Uses `prompt_influence: 0.3` for natural variety.
- **Music** (15s) — Country-specific musical style with regional instruments. Prefixed with "Musical piece:" for better SFX generation.
- **Language** (3-5 sentences) — A full conversational paragraph in the local language, generated with a dynamically created per-language voice via ElevenLabs voice generation API using `eleven_multilingual_v2`.

Location prompts are generated by **Backboard AI** with strict rules: no city names, landmark names, or station names in any audio prompt to prevent trivial identification.

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Setup

```bash
# Install dependencies
npm install

# Copy environment variables
cp .env.example .env.local
```

Add your API keys to `.env.local`:

```env
BACKBOARD_API_KEY=your_backboard_api_key
BACKBOARD_API_URL=https://app.backboard.io/api
ELEVENLABS_API_KEY=your_elevenlabs_api_key
```

### Run

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
npm start
```

## API Routes

| Route | Method | Description |
|---|---|---|
| `/api/game/generate` | POST | Generate a random location with 3 audio clues |
| `/api/game/guess` | POST | Submit map coordinates, receive score and actual location |
| `/api/leaderboard` | GET | Fetch top 100 scores |
| `/api/leaderboard` | POST | Save a score to the leaderboard |
