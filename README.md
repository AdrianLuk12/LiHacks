# SonicGuessr

A Geoguessr clone where players guess locations based on **AI-generated audio** instead of street view images. Listen to ambient sounds, local music, and spoken language -- then drop a pin on the map.

**Stack:** TypeScript + Express (backend) | React + Vite (frontend) | Backboard/Gemini (AI text) | ElevenLabs (AI audio) | Google Maps JS API

---

## How It Works

1. A seed script generates 3 rounds, each tied to a real city
2. For each city, AI produces 3 audio clips: ambient noise, local music, and a spoken phrase in the native language
3. Players listen to the clips, place a marker on a stripped-down map (no labels, no POIs), and submit their guess
4. The server scores the guess (0-5000 points) based on distance, reveals the answer, and roasts the player

---

## Game Loop

```
Start Game (3 rounds)
  |
  v
Round N:
  [1] Player sees a blank dark map
  [2] Player clicks "Ambient", "Music", or "Language" to hear audio clips
  [3] Audio visualizer shows animated frequency bars while playing
  [4] Player clicks the map to place a guess marker (draggable)
  [5] Player clicks "Submit Guess"
  [6] Server calculates haversine distance -> scores 0-5000
  [7] Server generates a sarcastic AI roast of the guess (stretch)
  [8] Map reveals: correct location marker + line from guess to target
  [9] Player clicks "Next Round"
  |
  v
After 3 rounds: game complete, total score shown
```

---

## 3 Audio Clips Per Round

Each round gives the player 3 distinct clues to make the location guessable:

| Clip | Generated By | Example (Paris) |
|------|-------------|-----------------|
| **Ambient** | ElevenLabs SFX from Backboard description | Busy cafe, clinking glasses, distant traffic, Vespa horns |
| **Music** | ElevenLabs SFX from Backboard description | Street accordion playing a French waltz near the Seine |
| **Language** | ElevenLabs TTS from Backboard phrase | "Bienvenue a Paris, le prochain metro arrive dans deux minutes" |

## API Contract

### `GET /api/round?round_index=0`

Returns audio URLs for a round. **Does not expose the answer (lat/lng).**

```json
{
  "audio": {
    "ambient": "/static/audio/round_1_ambient.mp3",
    "music": "/static/audio/round_1_music.mp3",
    "language": "/static/audio/round_1_language.mp3"
  },
  "round_index": 0,
  "total_rounds": 3
}
```

### `POST /api/guess`

Accepts a guess, returns score + answer.

**Request:**
```json
{
  "lat": 51.5074,
  "lng": -0.1278,
  "round_index": 0
}
```

**Response:**
```json
{
  "score": 1842,
  "distance_km": 341.2,
  "target_lat": 48.8566,
  "target_lng": 2.3522,
  "city": "Paris",
  "message": "Ah yes, London. Close enough if you squint at a map from space."
}
```

**Scoring formula:** `score = max(0, min(5000, round(5000 * exp(-distance_km / 200))))`

| Distance | Score |
|----------|-------|
| 0 km | 5000 |
| 50 km | 3894 |
| 200 km | 1839 |
| 500 km | 410 |
| 1000 km | 34 |

### Prerequisites

- Node.js 18+
- npm

```bash
# Server
cd server && npm install

# Client
cd client && npm install
```

```bash
cd server
npx tsx src/index.ts

# Terminal 2: Frontend (port 5173, proxies API to 3001)

npm run dev
```

**Haversine formula (implement inline, no extra package):**
```typescript
function haversine(lat1: number, lon1: number, lat2: number, lon2: number): number {
  const R = 6371;
  const dLat = (lat2 - lat1) * Math.PI / 180;
  const dLon = (lon2 - lon1) * Math.PI / 180;
  const a = Math.sin(dLat/2)**2 + Math.cos(lat1*Math.PI/180) * Math.cos(lat2*Math.PI/180) * Math.sin(dLon/2)**2;
  return R * 2 * Math.atan2(Math.sqrt(a), Math.sqrt(1-a));
}
```

**Key rules:**
- Person A: commit mock `game_data.json` as your **first** push so Person B isn't blocked

- Person B: owns `App.tsx` -- you are the integration point, merge C's components last
